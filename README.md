# Coding Challenge: CRUD Functionality for ManilaPayroll.com Master File

# Frontend Set up

Open client folder:

```bash
$ cd client
```

Install dependencies:

```bash
$ npm install
```

Run dev client:

```bash
$ npm run dev
```

# Backend Set up

Open server folder:

```bash
$ cd server
```
Install dependencies:

```bash
$ npm install
```

Run dev client:

```bash
$ npm run dev
```
